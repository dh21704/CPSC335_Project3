# Toy Arithmetic Expression Compiler (Lex & Yacc)

## Overview
This project implements a simple compiler front end using **Flex (Lex)** and **Bison (Yacc)** to recognize and evaluate arithmetic expressions generated by a given context-free grammar. The program acts as a calculator that evaluates valid expressions and reports errors for invalid ones.

## Supported Features
- Integer arithmetic
- Operators: `+`, `-`, `*`, `/`
- Unary minus (`-`)
- Parentheses for grouping
- Correct operator precedence and associativity
- Error detection and recovery (continues after invalid expressions)
- Division-by-zero detection

## Files
- `lex.l` – Lexer specification (Flex)
- `yacc.y` – Parser and evaluator specification (Bison)
- `pa2_test_cases.txt` – Sample input test cases
- `README.md` – Project documentation
